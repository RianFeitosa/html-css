Principais Formatações

Negrito / Destaque

Este termo está em negrito usando a tag <b>. (Forma não semântica)

Este termo está em destaque usando a tag <strong>. (Forma semântica)

Itálico / Ênfase

Este termo está em itálico usando a tag <i>. (Forma não semântica)

Este termo está em ênfase usando a tag <em>. (Forma semântica)

Texto marcado

Este termo está marcado usando a tag <mark>.

Texto grande e pequeno

Este termo está grande usando a tag <big>. (Forma não semântica e obsoleta)


Este termo está pequeno usando a tag <small>. (Forma não semâmntica)

Texto deletado

Este termo está marcado como deletado usando a tag <del>, indicando que deve ser lido, mas não considerado.

Texto inserido

Este termo está marcado como inserido usando a tag <ins>, indicando que ele foi adicionado depois. (Forma semântica)


Este termo está sublinhado usando a tag <u>.(Forma não semântica)

Texto subescrito
